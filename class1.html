Syntactically correct variable names

  • letters
  • numbers
  • dot or underline characters
  • starts with a letter
  • starts with a dot not followed by a number (e.g. ".2way" is not valid)
  • reserved words are not valid


Note: Rules for object names in R are flexible.

Stored objects are added to the workspace

Objects in the workspace (a.k.a. global environment) indicate objects stored in the current memory (i.e. in a server or RAM on your local computer for local installation of R)


R loads data and any object created to memory. Low memory space limits the data size which can be used.

Manipulate objects with functions


Generic syntax

function(<object>)

Find the average age

mean(age)

Functions often have optional arguments that can be specified and default options that can be changed

Objects have a class and other properties


Find the class of an object

class(<object>)

View other properties about an object

attributes(<object>)

str(<object>)

length(<object>)

dim(<object>)

Some properties may be NULL if they have never been set or are not relevant to the object

Functions to check the type of vector

is.numeric(<vector>)

is.character(<vector>)

is.logical(<vector>)

is.factor(<vector>)

Convert objects to a different type



Generic syntax

as.something(object)

Convert a numeric vector to a character vector

as.character(<vector>)

Functions are sensitive to object class


Output may differ depending on the object class


Or the function may not work at all with some objects
